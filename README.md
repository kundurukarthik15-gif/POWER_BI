# 📊 Power BI Dashboard Portfolio

Welcome to my Power BI project repository! 🚀

## 📌 Projects Included

### 🔹 1. Sales Dashboard

**Objective:**
Analyze sales performance and identify key revenue drivers.

**Key Insights:**

* Revenue trends over time
* Region-wise sales distribution
* Product performance analysis
* Top-selling categories

**Features:**

* Interactive filters (Region, Product, Time)
* KPI cards for quick insights
* Drill-down analysis

---

### 🔹 2. Customer Churn Analysis

**Objective:**
Identify patterns and reasons behind customer churn.

**Key Insights:**

* Churn rate and retention metrics
* Customer segmentation
* Factors influencing churn (age, usage, plan type)
* High-risk customer groups

**Features:**

* Predictive insights (if applied)
* Customer segmentation visuals
* Retention analysis dashboard

---

### 🔹 3. Marketing Funnel Analysis

**Objective:**
Track and optimize customer journey from awareness to conversion.

**Key Insights:**

* Conversion rates at each stage
* Drop-off points in the funnel
* Campaign performance
* Lead-to-customer ratio

**Features:**

* Funnel visualization
* Conversion tracking
* Campaign comparison
